1. Which CSS practice improves performance?
 A. Minifying CSS files
 B. Using fewer HTTP requests
 C. Using shorthand properties
 D. All of the above
 Answer: D. All of the above


2. Which reduces CSS render-blocking?
 A. Inline critical CSS
 B. Using @import for CSS
 C. Loading CSS at the end of body
 D. Using multiple CSS files
 Answer: A. Inline critical CSS


3. Which is better for performance: CSS sprites or individual images?
 A. CSS sprites
 B. Individual images
 C. Both same
 D. Depends on image format
 Answer: A. CSS sprites


4. Which property affects repaint and reflow heavily?
 A. width, height
 B. transform, opacity
 C. color
 D. font-family
 Answer: A. width, height


5. Which technique delays non-critical CSS?
 A. Lazy-loading CSS
 B. Using media queries
 C. Minifying CSS
 D. CSS variables
 Answer: A. Lazy-loading CSS


6. Using external CSS files improves caching. True or False?
 Answer: True


7. Inline styles improve or degrade performance?
 A. Improve
 B. Degrade
 Answer: B. Degrade (less reusable & cached)


8. Using !important impacts performance. True or False?
 Answer: False (affects maintainability, not performance)


9. Combining multiple CSS files into one improves performance. True or False?
 Answer: True


10. Which CSS unit is better for performance in animations?
 A. px
 B. em
 C. %
 D. rem
 Answer: A. px (less recalculation)
