
**Q1. CSS stands for:** 

A. Computer Style Sheets
B. Creative Style System
C. Cascading Style Sheets
D. Color Styling Syntax
**Answer: C**

**Q2. Which HTML tag is used to link an external CSS file?**
A. <style>
B. <link>
C. <script>
D. <css>
**Answer: B**

**Q3. Which attribute is used for inline CSS?**
A. css
B. style
C. design
D. inline
**Answer: B**

**Q4. Which of the following is NOT a type of CSS?**
A. Inline CSS
B. Internal CSS
C. External CSS
D. Embedded–External Hybrid CSS
**Answer: D**

**Q5. CSS is mainly used for:**
A. Structure
B. Functionality
C. Styling
D. Database
**Answer: C**

**Q6. External CSS uses which file extension?**
A. .html
B. .txt
C. .css
D. .js
**Answer: C**

**Q7. Which is the correct place to link external CSS?**
A. Inside <body>
B. Inside <head>
C. At the bottom of HTML
D. Anywhere
**Answer: B**


**Q8. The “Cascading” in CSS means:**
A. CSS is written in waterfalls
B. Styles are applied top-to-bottom
C. Styles depend on language
D. CSS always overrides HTML
**Answer: B**

**Q9. Which of these is fastest to load?**
A. Inline CSS
B. Internal CSS
C. External CSS
D. Database CSS
**Answer: C**

**Q10. Which CSS version is commonly used today?**
A. CSS1
B. CSS2
C. CSS3
D. CSS0
**Answer: C**
