## 2. CSS Selectors

**Q11. Which selector selects all `<p>` elements?**


A. .p
B. #p   
C. p  
D. *p  

**Answer: C**
##

**Q12. Which selector selects the element with id="title"?**


A. .title
B. #title
C. title
D. id.title

**Answer: B**
##

**Q13. Which selector targets all elements with class="box"?**


A. #box
B. .box
C. box
D. @box

**Answer: B**
##

**Q14. The universal selector is represented by:**


A. ?
B. /
C. *
D. %


**Answer: C**
##

**Q15. Which selector selects `<p>` elements with class="intro"?**


A. p.intro
B. intro.p
C. p#intro
D. intro>p

**Answer: A**
##



**Q16. `div > p` selects:**


A. All p elements inside div at any level
B. Only direct child p of div
C. p elements with class div
D. p elements in another file

**Answer: B**
##

**Q17. `a:hover` is an example of:**


A. Pseudo-element
B. Pseudo-class
C. Attribute selector
D. Universal selector

**Answer: B**
##

**Q18. Which selector selects all `<li>` inside `<ul>` at any depth?**


A. ul + li
B. ul > li
C. ul ~ li
D. ul li

**Answer: D**
##

**Q19. `[type="text"]` represents:**


A. Element selector
B. ID selector
C. Attribute selector
D. Class selector

**Answer: C**
## 

**Q20. `h1, h2, h3` is a:**


A. Group selector
B. Attribute selector
C. Universal selector
D. Class selector

**Answer: A**
##
